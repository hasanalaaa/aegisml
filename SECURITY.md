# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.x     | ✅ Yes    |
| 0.x     | ⚠️ Limited |

## Reporting a Vulnerability

**Please do not report security vulnerabilities via GitHub Issues.**

Email: security@aegisml.vercel.app

Include in your report:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (optional)

We respond within 48 hours and provide a fix within 7 days for critical issues.

## Scope

- AegisML web application (aegisml.vercel.app)
- AegisML API (scan engine)
- Python package (aegisml)
- GitHub Action
